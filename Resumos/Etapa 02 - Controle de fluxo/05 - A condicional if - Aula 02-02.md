# A condicional if - Aula 02-02

### Resumo

A condicional `if` é uma estrutura de controle de fluxo em JavaScript que permite que você execute uma série de instruções somente se uma determinada condição for verdadeira. A sintaxe básica é a seguinte:

```javascript
let number = 5;
if (number > 0) {
  console.log("O número é positivo");
}
```

Neste exemplo, a condição verifica se `number` é maior que 0. Se for verdadeira, a mensagem "O número é positivo" será exibida. Se for falsa, nada será exibido.

A condicional `if` é útil para algumas verificações como se um usuário está logado, se uma senha tem um determinado número de caracteres e outras possibilidades.
